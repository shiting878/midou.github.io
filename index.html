每个人生的阶段都是最美好的

发表于 2018-03-26   |   分类于 个人随笔   |  

我一直是个比较怀旧的人，从小的时候起就是这样。记得以前父亲常说：“开始怀旧了，人就老了。”那时我总打趣：“我也老了！”长辈们总是笑哈哈，说我还是小孩子就说自己老了。

时光流转得好快，童年和我亲爱的爷爷奶奶都风一般远去，不过他们活在我的记忆里，活在我的心里，也活在我的梦中，只要我还深深地记着他们，记着我们曾经共同有过的美好时光，他们就不是真正离我而去的。

今年过年前，大年二十八，姐姐生日那天，我掐指一算，姐姐已经三十二周岁了！我也即将二十七岁，而立之年本是书中读到的如此遥远的词，竟然兀地就在眼前了。美好的花季雨季，夕阳下奔跑着补考一千米的少年，如今都已正值青年啦。

想想往日的时光，在记忆的渲染下，分外美好，哪怕在当时是忧伤的、痛苦的、纠结的事情，现在回忆起来，都分外美好。幼儿园时的春游秋游很美好，背着小书包，装满爱吃的零食，和小朋友们排好队出发，一边唱着欢快的儿歌。中午就地野餐，吃得很是兴奋，最美好的是下午大概一两点，爸妈就来把我接回家，小朋友们也都开心地放半天假回家。记得有次春游完，老张也来我家，我们一起玩电脑游戏英雄无敌，那天下午天气很好，窗外阳光灿烂，我们一直玩到傍晚，他才依依不舍地回家。那时老张经常来家里玩，最早的射击类游戏三角洲，我们玩得爱不释手，两个人趴在地上，好像亲临战场，紧张时大气都不敢喘，静静埋伏在敌方的防御圈外。

说起玩游戏，我有很长的故事可以讲，每个阶段喜欢玩的游戏都不大一样，最经典的除了上面提到的英雄无敌和三角洲，还有更早期玩的魂斗罗，有即时战略游戏帝国时代、星际争霸，回合制游戏街头篮球、四国军棋、大富翁、角色扮演类游戏冒险岛、地下城与勇士、斩魂，手机音乐游戏节奏大师、Deemo等等。这些游戏我都曾玩得酣畅淋漓，自我感觉最擅长的游戏是玩了整整十多年的星际争霸和四国军棋，在大二时星际还一举斩获全国六强的成绩，四国军棋尤其擅长心理战，征战多年，敌军任何微小的细节都逃不过我犀利的双眼！然后就是大四开始玩的手机音乐游戏节奏大师，从开始玩的那天起，我几乎每天都练习两三个小时，从不间断，直到研二时一战成名，稳坐厦大软件学院第一把交椅。Deemo是我现在手机里唯一的游戏，我喜爱它干净简洁的黑白画风和纯静唯美的音乐体验，还有游戏时那份深深的沉醉的孤独感。

诚然，如今的游戏界五花八门，但我仍然钟爱那少数几款旷世经典的杰作，如果说只能有一款游戏被选为最爱，那我会毫不犹豫地回答——星际争霸。我在星际中感悟出很多生活的道理，也体会到战术战略的博大精深，玩得尽兴而且受益无穷，还有几个同样热爱星际的好友，这也是我一生宝贵的财富。

初高中的学习生活很美好，虽然在当时常处竞争氛围之中，天天都有做不完的习题，背不完的课文和写不完的作文。考试差了几分很惆怅，考好了又激动得忘乎所以。体育和历史总是差得要命，成天补考，呜呼哀哉。可现在想起来，都是青春岁月里几行宁静悠远的诗篇，刻在鸟儿叫唱的枝头，刻在洒满阳光的书桌，刻在花季雨季的画板。上周五晚上和女友在校园漫步，谈论起曾经那些数学和物理习题，仿佛又回到那段时光，也感叹全国中小学教育的内容竟是如此一致。

散漫文艺的本科生活实在是太美妙了，大连这座美丽的城市，有着凉爽宜人的气候，大连海事大学有大片充满温情的草场。那时我的心情有个成规律的曲线，每个月大概二十七八天都心情舒畅，也总有三两天情绪低落。情绪低落时往往食欲不佳，也无心学习，喜欢听着音乐在校园闲逛，或者回宿舍蒙头大睡一个下午。也许那时我就时常思考人生中那些永恒的没有答案的问题，想着想着便情绪低落，丢了平时发奋自习的韧劲，反思人生的意义和我向往的生活状态。多么美妙的时光啊，咖啡成了我日常读书学习的伴侣，偶尔也喜欢看教室里的美女，和几个室友一起，看她们安静学习的背影。大连的天空和大海都很蓝，校园的学子身上集合着大海的灵魂。

厦大读研的时光，那就更美了，校园本来就美，再加上校园里长发美女如云般游走在身旁，清新的空气中不时散发着淡淡的发香。湖泊和山光秀丽成趣，高尔夫球场自在挥杆，小琴房冬日凌晨独自练习钢琴，和女友成为音乐厅的常客，白龙马载着美丽心爱的梦，木棉树下心动的等待，夜里牵手的漫步。此中美意，无法文达，都化作温柔的晚风，拂过校园的一草一木。

现在的上班时光也是美好的，同事情谊在平日虽显无感，但风流云聚散，每当有熟识的同事离职，我都会默然伤感良久，一辈子的缘分也许就从此散去。即便平时工作中也有许多不快，也有烦心劳神的时候，也有拍案而起的时候，但终归是场浩大的缘分，是段不可重来的人生。

是啊，不可重来的人生，没有预演，也没有所谓的人生范式。生活是一天天过出来的，没有什么道路和规矩是一定正确，一定值得遵循的，任何事物都可以被重新设定，一切价值都可以被重新估算。我常常一边生活，一边在想之后什么样的生活才是值得度过的。我想不出，却喜欢常常去想，喜欢尝试新鲜的东西，读能带给我新的可能性的书，创作灵感出现的当下的诗篇，把握自己的时间和命运，不论我把时间用来驾驶，或者睡觉，或者只是简单地发呆。

锻造好自己这块材料吧，守护生活中值得珍惜的人和事，始终和我的兴趣爱好待在一起。我仍然有很多的困惑，我也仍然要继续走我的路。

被爱包围的童年——新春抒怀

发表于 2018-02-22   |   分类于 个人随笔   |  

刚过完春节，又回到熟悉的、几日不见的北辰山。一到山中便遇一场大雨，欢快的雨点落在脚旁，像是欢迎我的归来。中午冲咖啡的间隙，望了望远山和田野，绿油油一片，旺盛的生机丝毫没有被微冷的冬季遮掩。我像看一位许久不见的老友那样看着山野，别来无恙，很是欣喜。

这次回家是在除夕的前一天，请了半天假，提早一点回家与爸妈团聚。家乡年味不减，小小的城市依旧张灯结彩，祥和喜庆。什么是回家？回家就是亲人未见面时彼此的牵肠挂肚，回家就是背包里写满温馨祝福的两个红包，回家就是出了动车站便看见爸妈的身影就在出站口的转角，回家就是自己喜爱的食物总是满满当当地在桌上冒着热气，回家就是放下行李和琐事回归单纯日子的乐趣。

用一日清闲换内心平静

发表于 2018-01-25   |   分类于 个人随笔   |  

平日总感身体倍儿棒，不料近日偶感风寒，加上上班久了，没什么变化，只觉日子过得飞快，很多白昼美好的时光都没有好好珍惜就过去了，于是今个儿请了病假，也算工作日偷得一天安宁。

我和许多人休假的状态也许不同，有的人休假了还时不时要翻看邮箱，时不时查看工作微信群，担心有人找而没及时回复，会失去和错过些什么。我则不然，休假时心安理得，倍感欢愉，对工作上的事儿无牵无挂，工作时认真对待便是，下班或休假便完全隔绝，迅速回到自我最简单自在的生活状态中。这样也算劳逸结合，张弛有道，有的放矢吧。

我喜欢的清闲是什么呢？很简单，早上睡到自然醒，吃一顿美味的午餐，和心爱的人一起觅得一处海边咖啡屋，静静坐着发呆也好，读书也好，说话也好，就这样一直待到繁星满天，然后归去，睡个安稳的大觉！不多吃，不多想，不多关注其他外在的东西。我今天就是这样度过的，清静闲散的一天，找回内心的平静与真实。

失而复得的喜悦

发表于 2018-01-21   |   分类于 个人随笔   |  

最近日子过得比较浮躁，工作日繁琐的社交，心底涌动的欲望，亲戚无休止的谈话，上下班拥堵的翔安隧道，亲人间每日的情绪波动，以及无尽的衣袜和锅碗瓢盆，将岁月的诗意与安宁的时光冲刷殆尽。多浮躁的生活，书桌上竟不时落满尘埃，架子上的书没拆封的依旧没有拆封，细小的打扰可以轻易地中断灵动的思绪和萌生的诗意。

唉，不抱怨太多了，日子本就如此，繁繁复复，谁不是在夹缝和时间的碎片中寻找片刻自由？谁不渴望轻松自在地做自己？谁不想要说走就走的旅行？归根结底，一切都源自我们的比较心，源自无底的欲望，源自不安的灵魂。

日子一匆忙浮躁，诱惑一五光十色，记忆就脱轨。最近时常感觉自己对生活的记忆衰退得太多，常常记不起前两天都做了什么？这个月去了什么地方，几个周末是怎样度过的？今年都干了些什么，回了几次家？自己身上发生了什么重大的事情和难忘的成长？即便在当下用心体会，但是记忆却好像对生活愈加麻木与迟钝，记不住生活中很多值得珍惜的美好的细枝末节了，只有粗线条的，仿若做梦一般的印象。有时甚至分不清一件事情到底是梦中出现过的，还是现实生活里确实存在的。这种感觉真是太糟了！虽然我有时提醒自己要注意觉察，从旁观者的视角看看自己，但这份提醒本身，却泄露了我真正最缺少的、容易忽视的东西——当下的觉察。

当下的觉察少了，生活就容易陷入平庸与瞎忙，每天穿戴整齐出门，好像自己就像个人物，其实，这也许是我自己最鄙视的生活状态，靠外在的东西和无意义的充实来填充自己，这不正显示了自己的空白与无知吗？

时光里的诗

发表于 2018-01-07   |   分类于 诗歌   |  

岁月是一首唯美的诗
静默中
每个人都在雕刻自己生命的图景
不甘被现实打败的人们
总是倔强地背负着理想前行

那些燃烧的激情
和那懵懂青春里改变世界的坚定
稚嫩得常被嘲笑
却又多么可敬
在凌晨四点的街道守望黎明

多少明丽的感动
镌刻在时光的影像里
质朴平凡
像拾荒者的雨靴踩过泥泞
记忆中流淌音乐的生命

光被疾驰的列车模糊
夜晚灿烂而孤独的霓虹
日复一日
几代人的经营
一个家族的传承与信仰

岁月是一首唯美的诗
我们一边捕捞一边歌唱
同一个时代的徜徉
而每当你我忘乎所以
却总归还有些人在坚持着什么

大师为什么通常不是身边的人

发表于 2017-11-04   |   分类于 个人随笔   |  

坐在沙发上吃饼干发呆，茶几上摆着几本书，偶然看见最上面的一本，是聂鲁达的《我坦言我曾历尽沧桑》，封面的卡通简笔画上有个戴着帽子、嘴里正抽一支烟的形象，和聂鲁达很像，我曾在网上查阅他的资料，看过他的照片。不觉看得入神，脑中浮现出一个问题：为什么我认为是大师的人通常都不是我身边的人？

在想这个问题时，毋庸置疑，我觉得聂鲁达是伟大的诗人。我前不久读过他的《二十首情诗和一首绝望的歌》、《船长的诗》、《一百首爱的十四行诗》这几部诗集，最近正在读他的《疑问集》，名曰“疑问”，实际上还是由许许多多充满想象的小诗句构成的集子。他的情诗是我读到过最好的情诗，唯美的词语，适时的顿挫，无边的想象，滔滔不绝的赞美，乍一读，词句纷飞好似漫天棉絮，无着无落，仔细一想，内里还是有很多明明暗暗的联系。我每次读他的诗集，都舍不得多读，小读两三篇，便觉心满意足，我脑中想象力的马匹也跟着欢腾奔跑起来。聂鲁达是我心中的大师，可他是生活在上个世纪的人，离我很远，不是我身边的人。

我又想，还有哪些是我一下子就想得到的大师呢？叶企孙、尼采、爱因斯坦、米开朗基罗、鲁宾斯坦、莫扎特、贝多芬、肖邦、苏格拉底、柏拉图、亚里士多德、霍洛维茨、钱穆、陈寅恪、蔡元培、季羡林、胡适、克里希那穆提、弗洛伊德、普吕多姆、兰姆、伍尔芙、阿狄生、巴赫、苏东坡、北岛、潘懋元、周昌乐、钱兰英、马克西姆、理查德·克莱德曼、李云迪、罗素，能一下子浮现在脑海中的大致就这么些名字。我能想到的，多半是我有一定了解的人，要么是读过他们的书或传记，要么是常常听他们弹奏的音乐，要么是我所喜爱的作曲家、心理学家、教育家、文学家和诗人。对了，还有几位大师确实就在我身边，他们曾是我的老师——可敬的潘懋元老先生，精通各门学问的周昌乐老师，和心理学大师钱兰英老师。
